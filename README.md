## Installation

Just add it to your ``rebar.config`` deps:

```erlang
  {deps, [
    ...
    {pattern-match-util, ".*",
      {git, "git@github.com:NobukazuHanada//pattern-match-util.git", "master"}}
      ]}.
```

And then do the usual:

```bash
    $ rebar get-deps
    $ rebar compile
```
